# SudokuAI

Interactive Sudoku Solver and Generator built using Python and Pygame.

## Features
- Animated Backtracking Visualization
- Dark Mode UI
- Keyboard Input Support
- Sudoku Generator
- Timer Feature
- Solve Button
- Interactive Gameplay

## Technologies Used
- Python
- Pygame
- Git & GitHub

## Run Project

bash python3 main.py
